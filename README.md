# Weather App

## Description
A Python application that fetches live weather information using the OpenWeatherMap API.

## Features
- Search weather by city
- Temperature
- Humidity
- Weather condition
- Wind speed

## Technologies Used
- Python 3
- requests
- OpenWeatherMap API

## Installation

```bash
pip install requests
```

## How to Run

```bash
python weather_app.py
```

## Configuration

Replace the API key inside the code:

```python
API_KEY = "YOUR_API_KEY"
```
